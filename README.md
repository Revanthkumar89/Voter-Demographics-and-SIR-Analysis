# Voter-Demographics-and-SIR-Analysis
This project provides a data-driven view of voter records, highlighting key issues such as: Deceased voters still present in records Voters who have shifted or are absent Duplicate or multiple enrollments It also offers a detailed breakdown of voter demographics including gender distribution and district-wise analysis
Objectives: 
Identify and analyze inconsistencies in voter data
Support electoral roll cleaning and validation
Deliver actionable insights using visualization
📊 Dashboard Highlights
🔹 Voter Demographics
Total Voters: 109M+
Female Voters: 55M
Male Voters: 53M
Third Gender Voters: 14K
🔹 SIR Analysis
Deceased Voters: 8M (~27%)
Shifted Voters: 20M (~68%)
Multiple Enrollments: 1M (~4%)
🔹 Visualizations Used
Donut Charts (SIR & Gender Analysis)
Bar Charts (District-wise distribution)
Tables (Constituency-level insights)
Filters/Slicers (District & Assembly Constituency)
🛠️ Tools & Technologies
Power BI – Dashboard creation
DAX (Data Analysis Expressions) – Measures & calculations
Excel / CSV – Data preprocessing
📈 Key Insights
A large proportion of voters are marked as shifted, indicating migration trends
Significant number of deceased voters remain in records, highlighting data gaps
Small but important percentage of duplicate entries detected
Gender distribution is nearly balanced across the state
